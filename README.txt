A personal project building a top down Pokemon style game using JavaScript. 

To start game click on the canvas. 

Deployed Project: https://app.netlify.com/sites/heroic-crepe-916e2c/overview

This project was built using the following tutorial for guidance: https://www.youtube.com/watch?v=yP5DKzriqXA 
My focus for this project was to improve scalability and to gain a greater understanding of the differences between functional programming and OOP.
